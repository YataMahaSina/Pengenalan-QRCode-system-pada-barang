# Pengenalan-QRCode-system-pada-barang
pengenalan QRCode system pada barang merupakan rancangan prototipe yang digunakan untuk dapat melakukan proses tracking barang masuk serta keluar pada industri pemilahan barangn dengan menggunakan metode opencv serta dilengkapi dengan menu yang dapat bekerja untuk memilih sistem serta semua data dari proses disimpan pada database firebase
